# Snake Game

Simple Snake game built with Python's turtle module.

Features
- Move the snake with arrow keys.
- Eat food to grow and increase your score.
- Difficulty and screen size options on startup.

How to run
1. Ensure you have Python 3.8+ installed.
2. From the project folder run:

   python main.py

3. When prompted, enter a screen size (e.g., 600) and choose a difficulty (Easy, Medium, Hard).
